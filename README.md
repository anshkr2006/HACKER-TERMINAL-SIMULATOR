#  Hacker Terminal Simulator

> Hack. Decode. Survive.  
> An immersive cyberpunk-style hacker simulation game.
> 
#  Project Overview

Hacker Terminal Simulator is a multi-stage cyberpunk-inspired hacking game designed to provide an interactive and immersive gameplay experience through cybersecurity-based challenges and terminal-style simulations.

The game combines entertainment, problem-solving, and cybersecurity-inspired concepts such as:

- Password Cracking
- Cipher Decoding
- Binary Translation
- Morse Code Challenges
- Logic Gate Puzzles
- Memory Hacks
- Firewall Bypass
- Command Terminal Simulation
- XP & Rank System
- Black Market Shop
- AI Assistant
- Daily Challenges
- Animated Cyberpunk UI

#  Features

##  Password Firewall
Simulated password cracking and firewall bypass system.

##  Cipher Challenges
Includes:
- Caesar Cipher
- Vigenere Cipher
- Reverse Cipher

##  Terminal Command System
Interactive hacker-style command terminal.

##  Network Scan
Simulated port scanning and network security system.

##  Puzzle System
Different puzzle types:
- Binary Translation
- Morse Code
- Pattern Prediction
- Logic Gates
- Memory Hack
- Speed Typing

##  XP & Rank System
Level progression and hacker ranking system.

##  Black Market Shop
Unlock hacking tools and upgrades using earned credits.

##  AI Assistant
Interactive AI helper for guidance and hints.

##  Cyberpunk UI
Immersive terminal interface with:
- Neon glow effects
- Typing animations
- Glitch effects
- Matrix-style visuals
